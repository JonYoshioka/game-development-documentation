# Blender Shortcuts

## General Navigation & Interface

| Shortcut | Description |
|----------|------------|
| **Middle Mouse Button** | Rotate view |
| **Shift + Middle Mouse** | Pan view |
| **Scroll Wheel** | Zoom in/out |
| **Numpad 1** | Front view |
| **Numpad 3** | Right view |
| **Numpad 7** | Top view |
| **Numpad 9** | Flip view (opposite) |
| **Numpad 5** | Toggle orthographic/perspective |
| **Numpad 0** | Camera view |
| **Numpad Period (.)** | Focus on selected object |
| **Home** | Frame all objects |

## File Operations

| Shortcut | Description |

| **Ctrl + S** | Save file |
| **Ctrl + Shift + S** | Save as |
| **Ctrl + Alt + S** | Save copy |
| **Ctrl + Z** | Undo |
| **Ctrl + Shift + Z** | Redo |

## Transform Operations

| Shortcut | Description |
|----------|------------|
| **G** | Grab/move |
| **R** | Rotate |
| **S** | Scale |
| **Shift + S** | Snap menu |
| **Tab** | Toggle Edit/Object mode |
| **Shift + A** | Add menu |
| **X** or **Delete** | Delete menu |
| **Ctrl + A** | Apply transformations menu |
| **Ctrl + J** | Join selected objects |
| **P** | Separate (in Edit mode) |
| **Alt + D** | Duplicate linked |
| **Shift + D** | Duplicate |

| **Shift + Y** | Constrain to XZ plane |
