# Drawing Grid Guide

1. Select the **Actions Menu**
![Actions Menu](./assets/actions-menu.png)

2. Find the drawing guide items
![Drawing Guide](./assets/drawing-guide.png)

3. Enable **Drawing Guide** to get the grid then select **Edit Drawing Guide**
![Enable Drawing Guide](./assets/enable-drawing-guide.png)

4. In the **Drawing Guide** menu you can adjust the necessary settings
![Edit Drawing Guide](./assets/edit-drawing-guide.png)
