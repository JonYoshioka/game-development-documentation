# Creating a Grid Pattern

1. Select the **Selections Menu**
![Selections Menu](./assets/selections-menu.png)

2. Select **Rectangle** and ensure that **Add** and **Color Fill** are enabled
![Rectangle Settings](./assets/rectangle-settings.png)

3. Drag and create the rectangle
![Create Rectangle](./assets/create-rectangle.png)

4. Select the **Transform Menu** and set up snapping
![Snap settings](./assets/adjust-snap-settings.png)

5. Drag and adjust the rectangle to your preferred size
![Show Snapping](./assets/show-snapping.png)

6. Select the **Layers** 
![Open Layers](./assets/open-layers.png)

7. Swipe left on the layer to duplicate it
![Duplicate Layers](./assets/duplicate-layers.png)

8. The layers will be duplicated and now edit the new layer
![Duplicated Layers](./assets/duplicated-layers.png)

9. Use the **Rotate 45Â°** to make it perpendicular to the original line
![Rotate Layer](./assets/rotate-layer.png)

11. Pinch the two layers to combine into one layer
![Pinch to Combine](./assets/pinch-to-combine.png)

12. The final layer should look like this. You can duplicate it to make the pattern larger
![Combined Layers](./assets/combined-layers.png)
