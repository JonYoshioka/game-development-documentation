# Creating a Vite Project

## Quick Start
Create a new project

```sh
# Using npm
npm create vite@latest [project-name]
```

### Options that will be available:
You can choose a framework, the most used framework is `React`
```sh
◆  Select a framework:
│  ○ Vanilla
│  ○ Vue
│  ● React
│  ○ Preact
│  ○ Lit
│  ○ Svelte
│  ○ Solid
│  ○ Qwik
│  ○ Angular
│  ○ Marko
│  ○ Others
└
```

Next you can choose a variant, `Typescript` is a common language that is used
```sh
◆  Select a variant:
│  ● TypeScript
│  ○ TypeScript + SWC
│  ○ JavaScript
│  ○ JavaScript + SWC
│  ○ React Router v7 ↗
│  ○ TanStack Router ↗
│  ○ RedwoodSDK ↗
│  ○ RSC ↗
└
```
> `SWC` stands for Speedy Web Compiler and usually used for large codebases.

## Navigate and install dependencies
```sh
cd my-project
npm install
```

## Start development server
```sh
npm run dev
```
